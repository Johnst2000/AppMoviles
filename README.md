# Práctica 3 - Controles UI Android

Aplicación móvil nativa Android desarrollada como práctica académica.

## Descripción
Formulario de inscripción de usuarios con los principales controles 
de la Interfaz Gráfica de Android y estilos de Material Design.

## Controles utilizados
- TextInputLayout (Material Design)
- TextInputEditText
- ScrollView
- RadioGroup / RadioButton
- DatePickerDialog
- Button (Material)
- ConstraintLayout / LinearLayout

## Pantallas
- Pantalla de Inicio
- Inicio de Sesión
- Registro de Usuario (formulario completo con validaciones)
- Pantalla de Datos (muestra los datos ingresados)

## Tecnologías
- Android Studio
- Kotlin / Java
- Material Design Components
